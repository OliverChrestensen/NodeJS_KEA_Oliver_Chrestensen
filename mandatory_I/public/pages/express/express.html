NodeJS Express Server

 Use the npm init command in the terminal to create a package.json file for your application

  npm init

 Now you can edit what you would like to edit, or simply press enter until the file has been created.

 You might want to change index.js to app.js, but it's up to you.


 When you have created a working app you can run it by writing:

  node app.js

 Or whatever you chosed to called your JavaScript file.

To use express you have to install it first. Type this in the terminal:

  npm install express

now you have express installed and need to import it, chose a port for it to run on and make a simple route.

Here is a simple code snippet of how to make a simple express project

const express = require('express');
const app = express();

const port = 3000;

app.get('/', (req, res) => {
res.send('Hello World!');
});

app.listen(port, () => {
console.log(`Example app listening at http://localhost:${port}`);
});
